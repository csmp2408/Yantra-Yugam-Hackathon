IteraGen is a full‑stack prototype that transforms user prompts into interactive app scaffolds. Built with Node.js, Express, MongoDB, and a modern HTML/CSS/JS frontend, it showcases a complete workflow from idea to preview, trust analysis, and comparison.
✨ Features
- Sign‑In Dashboard → Secure login/register with JWT.
- Prompt‑to‑App Generation → Enter an idea, preview a scaffold (currently using dummy data; ready for TRAE API integration).
- Trust Meter & Diagrams → Circular gauge, radar chart, and bar chart to visualize app reliability.
- Comparison Mode → Side‑by‑side previews of two versions with trust delta and stability forecast.
- Library & Achievements → Manage generated apps, earn badges (First Deploy, Speedy Creator, Innovator).
- Profile & Feedback → Track trust scores, submit feedback, and view generation history.
🚀 Tech Stack
- Frontend: HTML, CSS, vanilla JavaScript
- Backend: Node.js + Express
- Database: MongoDB
- Deployment Ready: npm install → npm start
📝 Notes
- Current demo uses dummy scaffold data for app previews.
- Full integration with ByteDance TRAE API requires a paid developer plan.
- Architecture is already wired — once API credentials are added, the app will generate live code automatically.
